## What

Mozart is a **local-first, LLM-agnostic** cockpit for running coding agents in
parallel. Each agent works in its own isolated Git worktree, so you can conduct
several changes at once and review them as normal GitHub pull requests.

## Why

- **Isolated workspaces** — every agent runs in its own Git worktree. No agent steps on another's work.
- **Parallel agents** — launch many at once. They run simultaneously, not in a queue.
- **GitHub PR review** — ship through pull requests you already know. No new review surface to learn.
- **No cloud lock-in** — runs on your machine, with the model you choose. Your code never leaves.
